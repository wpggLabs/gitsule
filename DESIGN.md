# Design System

## Philosophy

Simple.

Fast.

Personal.

No enterprise dashboards.

No developer-tool complexity.

## Inspiration

- Obsidian
- Arc
- Raycast
- Readwise
- Pocket
- Steam Library

## Layout

### Sidebar

- Home
- Library
- Collections
- Favorites
- Settings

### Main Content

- Search
- Filters
- Repository cards
- Repository details

## Color Theme

Dark first.

Accent color:
GitHub blue.

## Components

### Repo Card

Shows:

- Name
- Description
- Stars
- Language
- Last Updated
- Status

### Collection Card

Shows:

- Collection name
- Repo count

### Status Badge

Options:

- Want To Try
- Testing
- Installed
- Favorite
- Abandoned

## UX Rules

- Everything searchable
- Keyboard friendly
- Fast local experience
- Offline-first mindset
