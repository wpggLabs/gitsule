# Gitsule

A capsule for your GitHub discoveries.

Gitsule helps you turn GitHub stars into a personal knowledge library.

Import starred repositories, organize them into collections, track what you've explored, add notes, and rediscover forgotten projects.

## Why?

Most developers have hundreds of starred repositories.

Over time:

- You forget why you starred them
- You never revisit them
- You lose track of what you've tested
- Interesting discoveries disappear into GitHub history

Gitsule solves that problem.

## Core Features

- Import GitHub starred repositories
- Personal repo library
- Collections and categories
- Status tracking
- Personal notes
- Search and filtering
- Rediscovery dashboard

## Status System

- Want To Try
- Testing
- Installed
- Favorite
- Abandoned

## Non Goals

Gitsule is NOT:

- A package manager
- A GitHub replacement
- A code hosting platform
- A developer analytics tool
- A social network

## Vision

Create a personal second brain for GitHub discoveries.
