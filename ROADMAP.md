# Gitsule Roadmap

## Phase 1 — Foundation

Goal:
Create a useful personal GitHub library.

Features:

- GitHub OAuth
- Import starred repos
- SQLite storage
- Repository list view
- Repository detail page

Success Metric:

- Import 100+ repos successfully

---

## Phase 2 — Organization

Features:

- Status system
- Collections
- Notes
- Tags
- Search

Success Metric:

- Use Gitsule daily for 30 days

---

## Phase 3 — Rediscovery

Features:

- Recently starred
- Forgotten repos
- Random discovery
- Activity insights

Success Metric:

- Rediscover at least 5 forgotten repos monthly

---

## Phase 4 — Exploration

Features:

- GitHub topic explorer
- Trending repositories
- Suggested repos

Success Metric:

- Discover new projects through Gitsule

---

## Phase 5 — Optional

Features:

- Launch recipes
- Install support
- GitHub release monitoring

Only build if personally needed.
